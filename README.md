## unroute me

Experiment on divergent search for map routes. Created during [Maquinando InteligÃªncias, Afetando Tecnologias](http://www.iea.usp.br/eventos/afetando-tecnologias-maquinando-inteligencias).


### Develop setup

```
$ git clone git@github.com:gecid-aia/unroute-me.git
$ cd unroute-me
$ pip install -r requirements.txt  # Using Python3
```

To run the app:

```
$ python app.py
```

Once the server is running, access it on [localhost:5000](http://localhost:5000/).
